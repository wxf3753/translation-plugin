🔧 快速设置

按照以下步骤开始使用：

1

获取Kimi API Key

访问 Kimi官网 注册账号并获取API Key

2

配置API Key

点击浏览器工具栏中的插件图标，在侧边栏中设置您的API Key

3

开始使用

选中网页文本，点击浮动按钮或使用快捷键进行翻译和总结

💡 关于Kimi API

Kimi是月之暗面公司开发的大语言模型，提供高质量的AI服务。使用前需要注册账号并获取API Key。插件会安全地存储您的API Key，仅用于与Kimi服务通信。

⌨️ 快捷键

提升效率的快捷键：

翻译选中内容 Ctrl + Shift + T
打开侧边栏 点击插件图标
快速翻译 Ctrl + Enter
